# 🌦️ WEATHER FORECASTING USING MACHINE LEARNING

> **An Intelligent Weather Prediction Web Application using Machine
> Learning, Flask and OpenWeather API**

## Project Overview

This project is an end-to-end Machine Learning weather prediction system
built with Python, Flask and Scikit-Learn. It combines historical
weather data, a trained ML model, real-time weather information from the
OpenWeather API and a responsive web interface with weather animations.

## Objectives

-   Predict weather using ML.
-   Deploy a trained model with Flask.
-   Display live weather information.
-   Provide an attractive UI with animated backgrounds.

## Workflow

1.  Load dataset.
2.  Train model.
3.  Save model and scaler.
4.  Load model in Flask.
5.  Receive user input.
6.  Scale features.
7.  Predict weather.
8.  Display prediction and live weather.
